# Taskify 📝

A simple task management app built with **Node.js** and ** JavaScript**.

## Features 🚀
- Add new tasks
- Edit existing tasks
- Delete tasks
- Toggle completion (optional)

## Tech Stack 💻
- Frontend: HTML, JS
- Backend: Express.js
- No database (in-memory storage)

## How to Run Locally 🛠️

### Backend:
```bash
cd backend
npm install
node server.js
